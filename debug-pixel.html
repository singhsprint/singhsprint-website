What to check in Meta Events Manager

  1. Open Events Manager → your Pixel → Test events tab.
  2. Enter your current page URL (e.g., https://singhsprint.com/debug-pixel.html) and click Open Website.
  3. Fire events using the buttons above.
  4. Each event should appear once with a "Received from" badge showing Browser + Server. That's successful dedupe.
  5. If you see two separate rows (one Browser, one Server) for the same event, the event_ids don't match — something is wrong with the pairing.
